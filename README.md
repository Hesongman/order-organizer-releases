# 订单整理工具

`订单整理工具` 是一个 Windows 桌面工具，用于批量整理订单 Excel 和 PDF。它可以读取订单行导出 Excel 中的订单信息，自动创建订单文件夹，移动对应文件，并生成站点清单汇总表。

## 下载

请到 Releases 页面下载最新版：

https://github.com/Hesongman/order-organizer-releases/releases/latest

下载后双击运行：

```text
订单整理工具.exe
```

## 主要功能

- 批量识别同一文件夹中的 Excel 和 PDF
- 按订单号匹配 Excel 与对应 PDF
- 自动创建 `订单号_订单名称` 文件夹
- 自动移动并重命名订单 Excel
- 生成 `站点清单汇总.xlsx`
- 支持只生成汇总表，不移动文件
- 支持拖入文件夹或 `.zip` 压缩包
- 启动时自动检查是否有新版本

## 使用方法

1. 把本批次的 Excel 和 PDF 放到同一个待处理文件夹。
2. 打开 `订单整理工具.exe`。
3. 在界面里选择 `输入文件夹` 和 `输出文件夹`。
4. 点击 `预检查`，确认可处理数量和跳过原因。
5. 确认无误后点击 `开始整理`。
6. 处理完成后，在输出文件夹查看订单文件夹和汇总表。

如果只需要汇总表，不想移动 Excel/PDF，请点击 `仅生成汇总`。

## 文件要求

Excel 需要包含以下工作表：

- `订单头数据`
- `订单行数据`

工具会读取：

- `订单头数据!B3`：订单号
- `订单头数据!A3`：订单名称
- `订单行数据` 中的站点、任务、金额、状态、项目等信息

PDF 文件名需要包含对应的订单号，工具才能自动匹配。

## 常见问题

### 找不到 PDF

请确认 PDF 文件名里包含 Excel `订单头数据!B3` 的订单号。

### 匹配到多个 PDF

说明多个 PDF 文件名都包含同一个订单号。请手动保留正确的 PDF，或修改文件名后重新预检查。

### 目标文件夹已存在

说明该订单可能已经整理过。工具默认不会覆盖已有文件，请先确认旧文件夹内容。

### 汇总表已经存在

如果输出文件夹里已经有 `站点清单汇总.xlsx`，工具会自动生成带时间戳的新汇总表，避免覆盖旧文件。

## 更新

工具启动时会自动检查是否有新版本。如果发现新版本，会提示打开下载页面。更新方式是重新下载最新版 `订单整理工具.exe` 后替换旧文件。
